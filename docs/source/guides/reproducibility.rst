Reproducibility Guide
=====================

Best Practices
--------------

1. Always use seeds for deterministic results
2. Store evidence packs with your results
3. Document parameter choices
4. Use invariants to catch errors early

Reproducing Results
-------------------

To reproduce results from an evidence pack:

1. Check ``manifest.json`` for Python and package versions
2. Install matching versions
3. Use the same parameters and initial state
4. Use the same seed
